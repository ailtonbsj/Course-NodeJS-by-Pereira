## Commands

```bash
# Install Express on .npm and show help
npx express-generator -h

# Create scaffold using EJS template engine
npx express-generator --view=ejs 05-introducao-express
cd 05-introducao-express

# Download modules
npm i

# Express Session
npm i express-session

# Express Method Override
npm i method-override

# Socket IO
npm i socket.io

# Consign
npm i consign
```
